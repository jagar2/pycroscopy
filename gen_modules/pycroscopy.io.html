class MicroDataset(name, data, dtype=None, compression=None, chunking=None, parent=None, resizable=False, maxshape=None)[source]¶

Bases: pycroscopy.io.microdata.MicroData

Holds data (i.e. numpy.ndarray) as well as instructions on writing, attributes, etc… This gets converted to a h5.Dataset by io.ioHDF5.

Region references need to be specified using the ‘labels’ attribute. See example below

Parameters:
  • name (String) – Name of the dataset
  • data (Object) – See supported objects in h5py
  • dtype (datatype,) – typically a datatype of a numpy array =None
  • compression ((Optional) String) – See h5py compression. Leave as ‘gzip’ as a default mode of compression
  • chunking ((Optional) tuple of ints) – Chunking in each dimension of the dataset. If not provided, default chunking is used by h5py when writing this dataset
  • parent ((Optional) String) – HDF5 path to the parent of this object. This value is overwritten when this dataset is made the child of a datagroup.
  • resizable (Boolean (Optional. default = False)) – Whether or not this dataset is expected to be resizeable. Note - if the dataset is resizable the specified maxsize will be ignored.
  • maxshape ((Optional) tuple of ints) – Maximum size in each axis this dataset is expected to be if this parameter is provided, io will ONLY allocate space. Make sure to specify the dtype appropriately. The provided data will be ignored

Examples

  1. Small auxillary datasets :
>>> ds_ex_efm = MicroDataset('Excitation_Waveform', np.arange(10))
  1. Datasets with slicing information :
>>> ds_spec_inds = MicroDataset('Spectroscopic_Indices', spec_ind_mat)
    ds_spec_inds.attrs['labels'] = {'Time Index':(slice(0,1), slice(None))}
  1. Initializing large primary datasets of known sizes :
>>> ds_raw_data = MicroDataset('Raw_Data', data=[], maxshape=(1024,16384), dtype=np.float16,
>>>                            chunking=(1,16384), compression='gzip')
  1. Intializing large datasets whose size is unknown in one or more dimensions:
>>> ds_raw_data = MicroDataset('Raw_Data', np.zeros(shape=(1,16384), dtype=np.complex64),
>>>                            chunking=(1,16384), resizable=True,compression='gzip')
