class FeatureExtractorParallel(detector_name, lib)[source]¶

Bases: object

This is an Object used to contain a data set and has methods to perform feature extraction on the data set that are detector based. Begin by loading a detector for features and a computer vision library.

Parameters:
  • detector_name ((string)) – name of detector.
  • lib ((string)) –

    computer vision library to use (opencv or skimage)

    The following can be used for: lib = opencv: SIFT, ORB, SURF lib = skimage: ORB, BRIEF, CENSURE

getFeatures(**kwargs)[source]¶

This is a Method that returns features (keypoints and descriptors) that are obtained by using the FeatureExtractor.Detector object.

  • mask (boolean, optional, default False.) – Whether to use
